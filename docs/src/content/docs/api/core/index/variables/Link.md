Anchor element that requires `href` and defaults to `target="_blank"`.

## Param

**props**

Link props.

## Param

**props.href**

Destination URL.

## Param

**props.target**

Link target. Defaults to `_blank`.

## Returns

An anchor element.

## Example

```tsx
<Link href="https://example.com/account">View your account</Link>
```
