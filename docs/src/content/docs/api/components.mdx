All components are imported from `hono-email`:

```ts
import { Html, Head, Body, Text, ... } from 'hono-email'
```

---

## Document structure

### `<Html>`

Root email document wrapper. Accepts all standard `<html>` attributes.

```tsx
<Html lang="en">
  <Head />
  <Body>Hello</Body>
</Html>
```

### `<Head>`

Container for `<title>`, `<style>`, `<Font>`, and other head elements. Styles generated by `hono/css` and `<Tailwind>` are normalized into this element.

### `<Body>`

Outer body wrapper. Accepts all standard `<body>` attributes.

```tsx
<Body style={{ backgroundColor: '#f6f9fc' }}>
  <Text>Hello</Text>
</Body>
```

### `<Preview>`

Renders hidden preview text that is relocated near the top of the email. Keep it short (under 200 characters) and place it immediately inside `<Body>`.

```tsx
<Html>
  <Preview>Your receipt is ready.</Preview>
  <Body>...</Body>
</Html>
```

## Layout

### `<Container>`

Full-width presentation table. Centers content and limits maximum width.

```tsx
<Container style={{ maxWidth: '560px', margin: '0 auto' }}>
  <Text>Hello</Text>
</Container>
```

### `<Section>`

Full-width presentation table whose direct children are table rows/cells.

```tsx
<Section>
  <Column width="50%">Left</Column>
  <Column width="50%">Right</Column>
</Section>
```

### `<Box>`

Generic email-safe wrapper.

#### `as` <Badge text="Optional" variant="note" size="small" />

Element tag to render. One of `div | span | table | tbody | tr | td`. Defaults to `div`.

### `<Card>`

Bordered table container with email-safe padding and background defaults.

#### `backgroundColor` <Badge text="Optional" variant="note" size="small" />

Cell background color. Defaults to `#ffffff`.

#### `borderColor` <Badge text="Optional" variant="note" size="small" />

Border color. Defaults to `#e5e7eb`.

#### `borderWidth` <Badge text="Optional" variant="note" size="small" />

Border width. Numbers are converted to pixels. Defaults to `1`.

#### `padding` <Badge text="Optional" variant="note" size="small" />

Cell padding. Numbers are converted to pixels. Defaults to `24`.

#### `contentStyle` <Badge text="Optional" variant="note" size="small" />

Inline style applied to the inner content cell.

#### `width` <Badge text="Optional" variant="note" size="small" />

Table width. Defaults to `100%`.

```tsx
<Card padding={24} borderColor="#e5e7eb">
  <Text>Plan updated.</Text>
</Card>
```

### `<Flex>`

Table-based flex-like layout. Does **not** use CSS `display: flex`.

#### `direction` <Badge text="Optional" variant="note" size="small" />

`'row'` (default) or `'column'`.

#### `align` <Badge text="Optional" variant="note" size="small" />

Vertical alignment for child cells: `'top'` (default) | `'middle'` | `'bottom'` | `'baseline'` | `'stretch'`.

#### `justify` <Badge text="Optional" variant="note" size="small" />

Horizontal table alignment: `'left'` | `'center'` | `'right'` | `'start'` | `'end'` | `'space-between'`.

#### `gap` <Badge text="Optional" variant="note" size="small" />

Space between children. Numbers are converted to pixels.

```tsx
<Flex align="middle" gap={12} justify="center">
  <Text>Left</Text>
  <Text>Right</Text>
</Flex>
```

### `<Grid>`

Table-based grid layout with a fixed number of columns. Does **not** use CSS `display: grid`.

#### `columns` <Badge text="Optional" variant="note" size="small" />

Number of columns per row. Defaults to `2`.

#### `align` <Badge text="Optional" variant="note" size="small" />

Vertical alignment for child cells. Defaults to `'top'`.

#### `gap` <Badge text="Optional" variant="note" size="small" />

Space between grid cells. Numbers are converted to pixels.

```tsx
<Grid columns={2} gap={16}>
  <Card><Text>One</Text></Card>
  <Card><Text>Two</Text></Card>
</Grid>
```

### `<Row>` / `<Column>` / `<Spacer>`

Table row (`<tr>`), table cell (`<td>`), and explicit spacing block helpers.

```tsx
<Row>
  <Column>Receipt total</Column>
  <Column>$42.00</Column>
</Row>

<Spacer height={24} />
```

`<Spacer>` props: `height` (default `16`) and `width` (default `100%`). Numbers are converted to pixels.

## Typography

### `<Text>`

Paragraph text. Defaults to `font-size: 14px`, `line-height: 24px`, and `margin: 16px 0`.

### `<Heading>`

Heading element with optional semantic level and margin shorthands.

#### `as` <Badge text="Optional" variant="note" size="small" />

Heading tag: `'h1'` (default) | `'h2'` | `'h3'` | `'h4'` | `'h5'` | `'h6'`.

#### Margin shorthands <Badge text="Optional" variant="note" size="small" />

`m`, `mx`, `my`, `mt`, `mr`, `mb`, `ml` â€” shorthand margin props. Numbers are converted to pixels.

```tsx
<Heading as="h2" mb={12}>Account summary</Heading>
```

### `<Link>`

Anchor element. Defaults to `target="_blank"`.

#### `href` <Badge text="Required" variant="caution" size="small" />

Destination URL.

### `<Button>`

Call-to-action link rendered with button-friendly defaults and Outlook padding support.

#### `href` <Badge text="Required" variant="caution" size="small" />

Destination URL.

```tsx
<Button
  href="https://example.com/start"
  style={{ backgroundColor: '#111827', color: '#ffffff', padding: '12px 16px' }}
>
  Get started
</Button>
```

### `<Img>`

Image element with email-safe defaults. Both `src` and `alt` are required.

#### `src` <Badge text="Required" variant="caution" size="small" />

Image URL.

#### `alt` <Badge text="Required" variant="caution" size="small" />

Alternative text for accessibility and clients that block images.

### `<Hr>`

Horizontal rule. Defaults to `border-top: 1px solid #eaeaea`.

### `<CodeInline>` / `<CodeBlock>`

Inline and block code with monospace font and light background defaults.

### `<List>` / `<ListItem>`

Ordered and unordered lists.

#### `ordered` <Badge text="Optional" variant="note" size="small" />

Render an `<ol>` when `true`, `<ul>` otherwise.

#### `marker` <Badge text="Optional" variant="note" size="small" />

CSS `list-style-type` value.

## Styling helpers

### `<Font>`

Renders `@font-face` and fallback font-family CSS inside `<Head>`. See [Font](/styling/font/).

### `<Tailwind>`

Applies a Tailwind build artifact to descendant class names. See [Tailwind](/styling/tailwind/).

#### `artifact` <Badge text="Optional" variant="note" size="small" />

Build artifact from `buildTailwindArtifactFromCss()` or injected by the bundler plugin. Required at runtime â€” if omitted, `render()` throws.

### `<ColorScheme>`

Declares light/dark color scheme preferences inside `<Head>`.

#### `colorScheme` <Badge text="Optional" variant="note" size="small" />

Declared color scheme. Defaults to `'light dark'`. One of `'light'` | `'dark'` | `'light dark'`.

#### `supportedColorSchemes` <Badge text="Optional" variant="note" size="small" />

Supported schemes meta value. Defaults to `colorScheme`.

### `<Conditional>`

Renders Outlook conditional comment branches.

#### `mso` <Badge text="Optional" variant="note" size="small" />

Render content inside `<!--[if mso]>...<![endif]-->`. Defaults to `true`.

#### `notMso` <Badge text="Optional" variant="note" size="small" />

Render content inside `<!--[if !mso]><!--> ... <!--<![endif]-->`.

### `<Markdown>`

Converts GFM Markdown into sanitized email-friendly HTML. See [Markdown](/markdown/overview/).
